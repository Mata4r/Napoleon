# Features
- Network sniffing
- Arp spoofing
- Ping
- Domain ip cache
- Domain ping

# Windows Installation
- Run CMD as Admin
- git clone https://github.com/Mata4r/Napoleon.git
- cd Ncap
- pip install scapy
- python Napoleon.py

# Linux Installation
- Run Terminal
- git clone https://github.com/Mata4r/Napoleon.git
- cd Ncap
- sudo apt update
- sudo apt install python3 python3-pip -y
- pip3 install scapy
- sudo python3 Napoleon.py
